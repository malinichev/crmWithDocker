# Simple CRM

## Deployment

Приложение распространяется как Docker-контейнер.

Пример:

```
docker compose up
```

Приложение внутри контейнера слушает `80` порт.

Данные параметры можно скорректировать в `.env`.

## Development

Приложение использует менеджер пакетов `npm`.

- `npm i`
- `npm run dev`
